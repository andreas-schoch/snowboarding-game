How to play
Premise Santas helpers lost all presents for the kids! Luckily Santa happens to be a pro-snowboarder and is able to collect them quickly while performing cool tricks. Keyboard controls Jump (press longer for higher jump) Balance and perform tricks (frontflip, backflip) Center yourself toggle pause Touchscreen controls Swipe up to jump Balance and flip by pressing left or right side of screen (Keyboard controls work much better atm) Combo System After each trick (backflip, frontflip) you have a few seconds to perform the next one to gain extra points. The combo counter will eventually run out if you stay on the ground for too long. If you crash, the combo points are lost. The points are only applied once the counter runs out or you reach the finish line.

About the game The game is released under the open-source MIT license: Source code. It is inspired by Alto's Adventure, iStunt and old flash based browser games. While initially intended as a small and simple christmas game for legartis that was built outside of working hours, the development will continue independently. The game will remain fully free and open-source. If you are interested to contribute feel free to contact Andreas Schoch. It would be awesome to allow people to create and upload their own custom levels, themes, characters etc. and build a little community around this game in the future.
